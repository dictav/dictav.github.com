Ferret という Apache Lucene を元にした Ruby での実装があるね。こいつを使うとドキュメントの全文検索ができるようになる。Namazu や Hyper Estraier なんかと同じ。ただ、Ferret はその仕組みであって Namazu や Hyper Estraier のようなアプリケーションじゃないんだ。これは自分のアプリに組み込むにはいいんだけど、どうやって使うかという点でとても迷う。それにテキスト以外のデータの取り扱いにも迷う。

最近、この Ferret を使った Rails アプリをみつけた。Picolena がそれ。MS-Office などのファイルもインデックスしてくれるみたい。インストールしてみたからメモ。

インストールは RubyGems で行うよ。

$sudo gem install picolena

これで

$picolena target_directory

とすれば、target_directory 中のファイルを人撮りなめて index を作成してくれる。ただ、Permission の関係で通常のユーザでは load でエラーになってしまう。

$ sudo chmod -R a+r /opt/local/lib/ruby/gems/1.8/gems/picolena-*

で応急処置。でも、通常ユーザから使えないのはセキュリティうんぬんのせい？？

あと、日本語には対応していないみたいだ。たぶん、MeCab と組み合わせる必要があるんじゃないかな。
