Objective-C を使っていると、Cocoa のメソッドを試してみたくなることがある。

#import <Foundation/Foundation.h>

int main(void)
{
    NSArray *array = [NSArray arrayWithObjects:@"hoge", @"piyo", nil];
    NSLog(@"%@", array);
    if([array writeToFile:@"/write/to/path" atomically:YES]){
        NSLog(@"success");
    }else{
        NSLog(@"fail");
    }
    return 0;
}

みたいな簡単なことを試すにもいちいちファイルを作成する必要があった。

そこで、このような用途には MacRuby を使ってみることにした。
