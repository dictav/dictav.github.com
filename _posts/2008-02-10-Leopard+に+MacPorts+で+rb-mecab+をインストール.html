MacPorts で Ruby 関連のものをインストールすると。Ruby も一緒にインストールされる。

Leopard では始めから Ruby も RubyGems も Rails も入っているので、これらがバッティングしてしまい不具合が生じる。

  1. MacPorts では Ruby 関連はインストールしない
  2. MacPorts で全部統一する

のどちらかにするのが良いようだ。

MacPorts で rb-mecab をインストールするから 2 でいく。

まずは MacPorts で Ruby 関連をインストール。

$ sudo port install ruby rb-rubygems rb-mecab

RubyGems をアップデート

$ sudo gem install rubygems-update
$ sudo gem update --system

準備完了。RubyGems で Rails その他をインストールする。

$ sudo gem install rails

$ sudo gem install sqlite3-ruby

$ sudo gem install ferret

$ sudo gem install fcgi

とりあえずこんなもんで動いてます。
