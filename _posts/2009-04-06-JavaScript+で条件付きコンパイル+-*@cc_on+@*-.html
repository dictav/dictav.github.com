--- title: JavaScript で条件付きコンパイル /*@cc_on @*/ date: 2009-04-06 tags: JavaScript layout: post ---

オライリーの JavaScript 本に「クロスブラウザなんて要するに IE のことだ」みたいに書かれている困ったちゃんな IE ですが、条件付きコンパイルというものがあります。

/*@cc_on
  @if(@_jscript)
IE だけが実行するコード
  @else*/
IE 以外が実行するコード
/*@end
@/

みたいにして使います。

/**/ の部分は他のブラウザからはただのコメントとして見えるので、コメント以外の部分が実行されるのです。

同じようなもので、HTML 用のものもありますね。

これを使えば IE 用に CSS を個別に設定できそうです。
