インストール

XULRunner は yum でインストールすることができた。他は上記サイトの通り。

yum install xulrunner xorg-x11-server-Xvfb xorg-x11-xauth ghostscript fonts-japanese
#Firefox をインストールする必要はない。

Gtk-WARNING **: Unknown paper size A4

どこの問題か良くわからないけど用紙サイズ周りでエラーが出る。mozilla2ps の -papername letter を指定してやればエラーは出なくなるので、出力された PS ファイルを ps2pdf -sPAPERSIE=a4 で用紙サイズを変更して PDF にすれば良い。

本当は mozilla2ps の -pdf true を使いたいのだけどね。
