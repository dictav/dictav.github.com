OpenGL ES 2.0 では、1.1 で提供されていた glFrustrum や glScale などの関数がなくなっていて困った。

OpenGL ES 2.0 からは GLSL を使ってプログラミングを行うので複数の Matrix を扱う事ができるから、なくなったのかな？？

なければ自分で作れば良い。GLSL では

attribute vec4 position;
uniform mat4 mvpMatrix;

void main()
{
	gl_Position =  mvpMatrix * position;
}

みたいにすれば簡単に行列の演算をやってくれるので、適当な Matrix を作って渡してやればいい。

行列の演算なんかは GLSL 側でやった方が簡単。でも、GLSL 側では attribute 変数や uniform 変数の数が決まっているので、簡単な演算は C の方でやる。

よし作るぞ！と思ったけど、あるものは使おう。
